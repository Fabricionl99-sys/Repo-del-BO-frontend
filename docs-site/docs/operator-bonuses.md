# Operator bonuses API

| Method | Path | Descripción |
|--------|------|-------------|
| GET | `/v1/operator-bonuses` | Listar bonos |
| POST | `/v1/operator-bonuses` | Crear bono |
| PATCH | `/v1/operator-bonuses/{bonus_id}` | Actualizar |
| DELETE | `/v1/operator-bonuses/{bonus_id}` | Eliminar |

Ver referencia interactiva en [app.social2game.com/docs/operator-bonuses](https://app.social2game.com/docs/operator-bonuses).
