# Social2Game API Examples

Public examples for operator integration. Copy your API key from the Back Office → Developers → API Keys.

## Node.js

```bash
cd nodejs
cp .env.example .env
npm install
npm run send-event
```

## Python

```bash
cd python
cp .env.example .env
pip install -r requirements.txt
python send_event.py
```

Docs: https://docs.social2game.com (or `/docs` in local dev).
