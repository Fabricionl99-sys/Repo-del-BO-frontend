# niveles · Backoffice Operador

Frontend React + TypeScript + Vite del Backoffice Operador de WINGOAT/niveles.

## Desarrollo

```bash
npm install
npm run dev
```

Copiar `.env.example` a `.env.local` para usar MSW en local.

## Scripts

- `npm run dev`: servidor Vite
- `npm run build`: type-check + build producción
- `npm run lint`: ESLint
- `npm run test:coverage`: Vitest con coverage
